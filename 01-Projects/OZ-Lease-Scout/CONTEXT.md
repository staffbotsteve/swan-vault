# OZ Lease Scout

**Company:** e2s Holdings LLC
**Type:** PropTech web application
**Status:** Active development

---

## What It Is

A web app that surfaces commercial real estate listings in Qualified Opportunity Zones, using Mapbox GL JS with Novogradac OZ tilesets. The goal is to make OZ investing accessible to smaller investors by combining zone eligibility data with live commercial listings.

## Technical Stack
- Mapbox GL JS for mapping
- Novogradac OZ tilesets for zone boundaries
- Listing data: **OPEN PROBLEM** — Crexi confirmed no API access; evaluating RealNex, LoopNet, and direct MLS feeds

## Intellectual Property

- Patent covers the method of overlaying OZ eligibility with commercial listing aggregation
