# OZ Lease Scout

**Company:** SwanBill LLC

PropTech web app for Opportunity Zone commercial real estate. Designed to replace LoopNet and Crexi for OZ investors. Active development with provisional patent filed.

## Key Details
- Scope: find retail/commercial space for sale or lease within Opportunity Zones
- Tenant matching tools (e.g., In-N-Out must be 11mi apart, Burlington 21mi apart)
- Features: demographics analysis, investor tools
- Tech: Mapbox GL JS, Novogradac OZ tilesets
- Key blocker: listing data sourcing — Crexi has no API
- IP: provisional patent filed
- Status: active development
