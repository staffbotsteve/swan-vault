# Project Falcon

Ed-tech Student Information System (SIS) designed to replace PowerSchool, Blackbaud, and the fragmented K-12 software ecosystem. Pilot partner is Christian Brothers High School (CBHS) in Sacramento, CA. Built with Lovable.

## Key Details
- Pilot school: Christian Brothers High School (CBHS), Sacramento
- Architecture: Kernel (10 core modules) + Add-ons (6 premium) + Integrations (15+ via API/LTI)
- Revenue model: $50–75K/yr base license per school
