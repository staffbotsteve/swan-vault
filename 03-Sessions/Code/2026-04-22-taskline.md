# Taskline — Linear clone scaffold

## Task
Build a Linear-style project management tool in Next.js with projects, tasks/issues, status, priority, and user assignment.

## Outcome
Shipped a working Linear clone at `~/project-folders/taskline/app`. The app runs on Next.js 16 + Prisma 7 + SQLite with a dark, monospace-flavored UI featuring projects, nested tasks/issues, inline status/priority/assignee editing, list+board views, command palette (⌘K), comments, labels, due dates, and a "My issues" personal view. Database is seeded with 4 users, 3 projects, 11 tasks, 6 labels for immediate usability.

## Decisions made
- Project name: **Taskline** (Next.js app sits at `taskline/app/`)
- SQLite + Prisma over Postgres for zero-config local dev
- Prisma 7 required: `prisma.config.ts` for datasource URL + `PrismaBetterSqlite3` adapter on every PrismaClient instance (different from older Prisma)
- Next.js 16 async `params` API used everywhere in route handlers
- Custom CSS variables (not Tailwind theme tokens) to match Linear's specific dark palette
- All status/priority icons hand-rolled SVG (Linear-style filled circles, bar volume indicators) — no icon lib
